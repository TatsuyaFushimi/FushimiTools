Flare Downloader

YouTube の URL を貼り付けて MP4 でダウンロード。
解像度を選択可能。ローカルで動作するので高速・安全。

macOS Apple Silicon AI不要 無料
ダウンロード
Flare Downloader をダウンロード

FlareDownloader-arm64.zip（Apple Silicon専用）

動作環境
Mac Apple Silicon（M1以降）
macOS Ventura 13.0 以降推奨
ffmpeg Homebrewでインストール必要
セットアップ手順
1
ffmpeg をインストール （チームフレアはFlarePocket導入時にインストール済み：スキップ）
Homebrew が未インストールの場合は先に brew.sh からインストール。
その後ターミナルで：
brew install ffmpeg
2
zip を解凍して .app を移動
ダウンロードした zip を解凍し、Flare Downloader.app を「アプリケーション」フォルダへ移動。
3
初回のみ：右クリック → 開く
署名なしアプリのため、初回起動は 右クリック（2本指クリック）→「開く」 を選択。
警告ダイアログが出たら「開く」をクリック。2回目以降は普通にダブルクリックで起動。
4
ブラウザが自動で開く
アプリを起動するとブラウザで自動的に localhost:5001 が開きます。
5
Cookie を設定（初回のみ）
YouTube の年齢制限や会員限定動画をダウンロードする場合は Cookie の設定が必要です。
アプリ内の「Cookie 設定」から手順に従ってアップロードしてください。
⚠️ アプリを終了するとダウンロードが中断されます。ダウンロード完了まで終了しないでください。
